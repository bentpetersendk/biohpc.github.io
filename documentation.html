Documentation

BioHPC utilizes the University of Copenhagen Rubus HPC platform.

Most technical documentation, including compute access, job submission, storage, software modules, and general HPC usage, is available through the Rubus documentation portal.

Technical documentation is available at rubus.ku.dk.

BioHPC-specific guidance focuses on onboarding, Project Spaces, user management, and support.
